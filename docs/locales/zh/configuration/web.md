# Web

## 设置

```yaml
######################
##### WEB CONFIG #####
######################

# 与网页模板和发送邮件通知等相关的配置

# 字符串。GoToSocial 尝试加载 HTML 模板 (.tmpl 文件) 的目录。
# 示例: ["/some/absolute/path/", "./relative/path/", "../../some/weird/path/"]
# 默认值: "./web/template/"
web-template-base-dir: "./web/template/"

# 字符串。GoToSocial 试图提供静态网页资源（图片，脚本）的目录。
# 示例: ["/some/absolute/path/", "./relative/path/", "../../some/weird/path/"]
# 默认值: "./web/assets/"
web-asset-base-dir: "./web/assets/"
```
